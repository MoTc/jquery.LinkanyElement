<h1>Link any Element Jquery Plugin </h1>

<b>LinkanyElement</b> is a simple Jquery Plugin to attach links to elements

<h2>Version</h2>
0.1.0

<h2>Usage</h2>
    jQuery('span').LinkanyElement({

          href: 'https://www.nvidia.com',
          text: 'Click me'
           
     });
